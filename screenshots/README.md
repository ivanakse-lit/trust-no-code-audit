# Screenshots Directory

This directory contains screenshots of the Trust No Code Audit Dashboard frontend.

## Required Screenshots

Please capture the following screenshots and save them in this directory:

1. **audit-config.png** - Main audit configuration screen showing:
   - Audit type selection (Security/Architecture/Production)
   - Codebase path input
   - Reports folder input
   - Copy to Clipboard button

2. **path-browser.png** - Path browser/file picker interface showing:
   - Drive/folder navigation
   - Path validation indicators

3. **report-viewer.png** - Report viewer showing:
   - List of available reports
   - Markdown rendering of a sample report
   - Syntax highlighting

4. **dashboard-overview.png** - Full dashboard view showing:
   - Navigation tabs (Configure, Reports)
   - Overall UI layout and dark theme

## How to Capture Screenshots

1. Ensure the frontend is running: `npm run dev` from the `frontend` directory
2. Open http://localhost:3100 in your browser
3. Navigate through the different sections
4. Use your OS screenshot tool:
   - **Windows**: Win + Shift + S
   - **Mac**: Cmd + Shift + 4
   - **Linux**: Use your distribution's screenshot tool
5. Save screenshots with the names listed above
6. Recommended size: 1920x1080 or similar high-resolution

## Screenshot Guidelines

- Use the dark theme (default)
- Show realistic example paths and data
- Ensure UI elements are clearly visible
- Crop to show relevant content without excessive whitespace
